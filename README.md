# Pr4Entornos - Proyecto de Cajero (Refactor + Git)

Este proyecto es una práctica de la asignatura Entornos de Desarrollo.

## 📌 Descripción
El objetivo del proyecto es trabajar con control de versiones utilizando Git, realizar refactorización de código y documentar clases con Javadoc.

Se trabaja con una aplicación Java que simula un cajero automático.

## 👥 Trabajo en equipo

- **Alumno A:**
  - Creación del repositorio
  - Configuración inicial del proyecto Maven
  - Añadido de .gitignore
  - Documentación completa con Javadoc en la clase Cajero

- **Alumno B:**
  - Refactorización de la clase Cajero
  - Rename de variables y métodos
  - Encapsulación de atributos
  - Introducción de constantes (IVA)
  - Extracción de métodos (refactor de cobrar)

## ⚙️ Tecnologías utilizadas
- Java
- Maven
- Git / GitHub

## 📂 Estructura del proyecto
- Producto.java
- Cajero.java
- Main.java

## 🎯 Objetivo
Practicar el uso de Git con ramas, merges, commits y resolución de conflictos en equipo.
